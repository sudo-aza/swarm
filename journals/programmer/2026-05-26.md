
## Turn 2026-05-26 00:00 UTC+8 — v3.36 Cross-session overlap prevention

### Task
Task #180: Fix BOTH ghost-narrowing AND body-text overlaps simultaneously.

### Approach
Added cross-session vspace check in `\swarmwrapnext`:
- When a new wrapping session starts, check if the previous session's
  figure (placed via `\smash{\rlap}`) extends past the current vertical
  position on the same page.
- If `pagetotal < fig_bottom` on the same page, add `\vspace` to push
  past the previous figure's visual zone.
- This prevents full-width text from overlapping with the still-visually-
  present previous figure.
