## 02:00 UTC+8 — Task #145: \swarmwrappenalty{N} option (swarmwrap.sty v3.6)

**Task**: Add a user-configurable penalty option to discourage page breaks at the parshape boundary (mitigating ghost narrowing).

**What was done**:
1. Added `\newcount\swarmwrap@penalty` (default 10000) and `\swarmwrappenalty{N}` user command.
2. Added `\newdimen\swarmwrap@tw@lua` register — Lua-readable copy of the wrapped text width, set by `\swarmwrapnext`.
3. Implemented Lua `swarmwrap_post_lb()` function registered via `luatexbase.add_to_callback("post_linebreak_filter", ...)`.
4. Callback walks the broken-line hlist nodes, identifies the last line with width <= (tw + 3pt tolerance), and inserts a penalty node after it.
5. Callback is a no-op when `\swarmwrap@tw@lua == 0` (no wrapping active) or penalty == 0.
6. Updated package header: version v3.6, added `\swarmwrappenalty{N}` to usage docs, updated Known Limitation #1 to document the mitigation.
