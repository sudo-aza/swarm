## 13:00 UTC+8 — Task #151: Inter-narrow penalty boosting (swarmwrap.sty v3.14)

**Task**: Mitigate ghost narrowing on continuation pages by boosting penalties between ALL consecutive narrow lines.

**Problem**: In the NORMAL path, when a wrapped paragraph spans a page break, parshape narrowing persists to the continuation page but the figure does not (it's placed via `\smash{\rlap{}}` which is page-local). v3.6's `\swarmwrappenalty{N}` only inserted a penalty at the boundary between the last narrow line and the first full-width line. TeX could freely break between any two narrow lines (default `\interlinepenalty=0`), causing many narrow continuation lines on the next page with no figure beside them.

**Approach**: Modified `swarmwrap_post_lb` in the `post_linebreak_filter` callback to boost ALL existing penalty nodes between consecutive narrow lines to `\swarmwrappenalty{N}` (default 10000). The callback now tracks `prev_was_narrow` state: when a penalty node is encountered after a narrow hlist, its penalty value is raised. The original boundary penalty insertion is retained as a safety net.

**Effect**: TeX is strongly discouraged from breaking anywhere within the narrow zone. It prefers to break before or after the entire narrow zone. This reduces ghost narrowing from potentially many continuation lines to 0-1 lines in most cases. Very long paragraphs that must span a page break may still have 1-2 narrow continuation lines, but this is significantly better than before.

**Changes**:
1. Rewrote `swarmwrap_post_lb` Lua function: added `prev_was_narrow` tracking, penalty boosting on existing inter-narrow penalty nodes.
2. Added early return when penalty_val==0 (prevents unnecessary processing).
3. Updated header: version v3.14, updated Known Limitation #1, added v3.14 changelog.
4. Updated penalty control comment to describe inter-narrow boosting.

**Results**: test-customwrap.tex (8pp, 0 errors), test-pagebreak-variations.tex (15pp, 0 errors). No regressions.
