### Code Changes
1. **swarmwrap.sty v3.59**: Added 3*baselineskip safety margin to DEFER check. Previously compared `remaining == nl*bs`, now compares `remaining < nl*bs + 3*bs`. This causes more figures near page bottoms to be safely deferred to the next page, preventing narrow-line carry-over. Simple but effective fix for 2 of 3 ghost-narrowing cases.
2. **detect-layout-issues.py v10**: Added vertical overlap filter for excessive narrowing. Old version counted ALL narrow lines to the left of a figure regardless of vertical position. Short paragraph last-lines ("libero.", "Cras") anywhere on the page were counted as "wrapping zone," inflating ratios to 4-9x. Fix: only count lines within figure's vertical range (+10pt margin). Result: 640→0 excessive narrowing (1000-fig), 29→0 (50-fig).
3. **detect-layout-issues.py v6.2**: Added multicol column detection for misaligned figures. Fixes 4 false positives where figures in narrow multicol left columns were flagged as misaligned.
4. **swarmwrap-callback.lua v3.60**: pcall guard for `tex.dimen["pagegoal"]` and `tex.dimen["pagetotal"]` access in needspace callback. Defensive coding — prevents Lua crash if dimen registers not accessible in certain LuaTeX builds.
5. **New test files**: `src/test-wrapfig/test-50fig.tex` (456 lines) and `test-1000fig.tex` (10006 lines). These are NOT the standard stress tests.
