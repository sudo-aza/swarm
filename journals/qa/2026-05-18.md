### API assessment

The new API is clean and simple:
```latex
\begin{swarmwrap}%
  \begin{minipage}[t]{3cm}
    \centering
    \includegraphics[width=\linewidth]{photo}
    \captionof{figure}{Caption.}
  \end{minipage}%
\end{swarmwrap}%
\swarmwrapnext
Text wraps here.
```

The parshape computation is now deferred to `\swarmwrapnext` (not in the environment end code), which means `\linewidth` is captured at the correct location — inside multicol, itemize, etc. This is architecturally cleaner than v2.5 where the computation happened at `\end{swarmwrap}`.

Left-wrap code fully removed: no `\ifswarmwrap@right`, no `\llap` branch, no `\swarmwrap@ind`. Only the rlap branch remains. Code reduced from 213 lines (v2.5) to 142 lines (v3.0).

### Rating: 10/10 — PASS
