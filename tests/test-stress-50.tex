% test-stress-50.tex — 50-figure stress test for swarmwrap.sty
% Compiles with: lualatex test-stress-50.tex
% Purpose: Stress-test swarmwrap with 50 consecutive figure placements
%   using varied figure heights and text amounts, with NO section breaks
%   between figures. This tests the core wrapping loop and exposes:
%   - body-text overlaps (parshape leak)
%   - ghost narrowing (parshape persistence across pages)
%   - hollow carry-over (empty narrow text regions)
%   - wrongful whitespace (near-empty pages)
