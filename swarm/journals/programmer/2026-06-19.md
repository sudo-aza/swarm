### Task #208: 1000-fig page count regression 183→200 pages

**Root cause**: The anti-waste squeeze check (v3.51) was inside the `\ifdim remaining < fh` block. When the 5th figure on each page had remaining < fh (because the 4th figure's wrapping text consumed remaining space), the check computed space-after and compared it against the threshold. Both v3.51's `\swarmwrap@fh` threshold and an attempted `\swarmwrap@squeezemin` threshold were too conservative — they prevented squeeze-fit that v3.49 used to achieve 6-fig packing.

**Fix (v3.53)**: Removed the anti-waste squeeze check entirely. The squeeze-fit path now runs whenever `remaining >= squeeze-factor*fh AND remaining >= squeezemin`, without the additional anti-waste gate. This restores v3.49 packing density.
