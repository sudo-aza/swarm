\section*{Test 1: Basic Figure Wrapping (Right)}
Text should flow around a figure placed on the right side.
floatflt uses \texttt{floatingfigure} environment with \verb|[r]{width}|.
Note: floatflt uses \texttt{\textbackslash everypar} hooks — the figure must
appear BEFORE the paragraph text that wraps around it.

\begin{floatingfigure}[r]{0.35\textwidth}
  \centering
  \rule{0.3\textwidth}{4cm}
  \caption{Right-wrapped figure (floatflt).}
  \label{fig:right}
\end{floatingfigure}
