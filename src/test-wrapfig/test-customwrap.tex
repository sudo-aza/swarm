%   text on the next page is also narrowed by parshape (which persists
%   across page breaks). The figure and text are aligned with ~14pt gap.
%   After the parshape lines are exhausted, text returns to full width.
%   Trade-off: narrowed lines on the current page have no figure beside
%   them (wasted horizontal space). This is far better than the v3.2
%   centered fallback which had zero wrapping on either page.
% ====================================================================
\section*{Test 6: Page Break Fallback}
This test verifies that swarmwrap.sty detects insufficient remaining
page space and triggers the RIGHT-WRAP fallback: parshape is applied
to the paragraph (persisting across page breaks). On the current page,
text is narrowed with no figure beside it. On the next page, the figure
is placed at the top-right and continuation text wraps around it.

% Fill exactly enough space to leave ~80pt remaining on this page.
\newdimen\swtestsixht
\swtestsixht=\textheight
\advance\swtestsixht by -80pt

  leaving approximately 80pt of remaining space. The swarmwrap
  figure requires more space than this, so the page break
  detection in \texttt{\textbackslash swarmwrapnext} should
  trigger the RIGHT-WRAP fallback: parshape narrows text on this
  page (no figure beside it), and on the next page the figure
  appears at the top-right with continuation text wrapping around it.%
