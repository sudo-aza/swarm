page space and triggers the fallback: text fills the rest of the
current page at full width (no wrapping), and the figure appears
CENTERED at the top of the next page via \texttt{\textbackslash afterpage}.

  figure requires more space than this, so the page break

  trigger the CENTERED fallback: text fills remaining space at
