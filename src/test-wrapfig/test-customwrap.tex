\section*{Test 1: Basic Figure Wrapping (Right)}
This test wraps a figure on the right side of the paragraph. The text should flow
around the image naturally, leaving a gap for the figure on the right margin.

\begin{swarmwrap}[r]{3cm}{4cm}
  \rule{\linewidth}{3.8cm}
  \captionof{figure}{Right-wrapped test figure (3cm wide, 4cm tall).}
  \label{fig:right}
\end{swarmwrap}

% Test 3: Tall Figure Near Page Break
% Expected: PASS for wrapping coverage. N/A for page-break spanning.
%   parshape cannot span page breaks (TeX resets per-paragraph).
%   The tall 8cm figure needs ~19 narrow lines. Using a single merged
%   paragraph (\lipsum[1-3] concatenated) to ensure all lines wrap.
% ====================================================================
\section*{Test 3: Tall Figure Near Page Break}
This test places a tall wrapped figure alongside a long single paragraph.
parshape-based wrappers cannot span page breaks by design. The text
below is a single merged paragraph (multiple lipsum paragraphs joined
with no blank lines) to ensure parshape applies to all lines.

\begin{swarmwrap}[r]{3cm}{8cm}
  \rule{\linewidth}{7.8cm}
  \captionof{figure}{Tall figure (8cm).}
  \label{fig:pagebreak}
\end{swarmwrap}
\swarmwrapnext
\lipsum[1]\lipsum[2]\lipsum[3]% No blank lines — single merged paragraph

% ====================================================================
% Test 4: Figure Inside Itemize (EXPECTED PARTIAL)
% This is a known hard constraint. Most packages fail here.
% swarmwrap uses \parshape which applies to the paragraph level,
% so \item boundaries may cause issues.
% ====================================================================
\section*{Test 4: Figure Before Itemize}
The supported approach: place \texttt{swarmwrap} BEFORE the list, so that
the wrapped paragraph text flows around the figure. List items after the
wrap region return to full width.

\begin{swarmwrap}[r]{3cm}{3cm}
  \rule{\linewidth}{2.8cm}
  \captionof{figure}{Figure before itemize (3cm).}
  \label{fig:before-itemize}
\end{swarmwrap}
\swarmwrapnext
\lipsum[1]

  \item First item: \lipsum[2][1-3]

  \item Third item: \lipsum[4][1-3]
  \item Fourth item: \lipsum[5][1-3]
  \item Fifth item: \lipsum[6][1-3]
\end{itemize}

\section*{Test 6: Inside Multicol}
Inside multicol, swarmwrap applies parshape using the column-width
the column-width textwidth — this is a known limitation.
