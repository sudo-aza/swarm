# Research Notes — 2026-05-14

## Portable TeX Live Installation

- Use `install-tl --portable --scheme=full --no-interaction` for standalone install
- `--portable` flag is key — disables post-install folder creation
- Install to project directory: `./texlive/2025/`
- Add to PATH: `export PATH="./texlive/2025/bin/x86_64-linux:$PATH"`
- Schemes: `scheme-full` (~8-10 GB), `scheme-medium` (~4-5 GB), `scheme-small` (~2-3 GB)
- Post-install package management: `tlmgr install <pkg>` / `tlmgr update --self --all`

## Best Modern LaTeX Themes (2024-2026)

### Beamer
- **Moloch** (v2.0.0+) — successor to Metropolis, actively maintained
- Metropolis is legacy but still widely used
- Moloch highlighted at TUG 2024

### Article/Book
- **KOMA-Script** (v3.49) — `scrartcl`, `scrbook`, `scrreprt` — replaces standard classes
- **Tufte-LaTeX** (v3.5.2) — side notes, margin figures
- **ModernCV** (v2.1.3) — for resumes

### Must-have packages
- `fontspec` (OpenType fonts), `microtype` (microtypography), `unicode-math`
- `tcolorbox` (boxes, replaces mdframed), `booktabs` (tables), `tabularray` (modern tables)
- `biblatex` + `biber` (bibliography), `tikz` + `pgfplots` (graphics)
- `minted` or `listings` (code highlighting)
- `hyperref`, `xcolor`, `amsmath`, `mathtools`

## LuaLaTeX Performance Measurement

### Compilation timing
```latex
callback.register("stop_run", function()
    texio.write_nl(string.format("\n[PERF] %.3f seconds\n", os.clock() - start))
end)
```

### PDF size
```lua
callback.register("finish_pdffile", function()
    local f = io.open(tex.jobname .. ".pdf", "rb")
    local size = f:seek("end")
    f:close()
    -- report size
end)
```

### Word count
- External: `texcount document.tex` (gold standard)
- Internal: Lua `process_input_buffer` callback (heuristic)

### Key callbacks
- `start_run` / `stop_run` — total compilation time
- `finish_pdffile` — final PDF size
- `start_page_number` / `stop_page_number` — per-page timing
- `pre_linebreak_filter` / `post_linebreak_filter` — paragraph metrics

### Performance note
- LuaLaTeX is ~2-5x slower than pdfLaTeX for typical documents
- `luaotfload` font caching improves performance after first run

## Syntax Highlighting

### minted vs listings
| Feature | minted | listings |
|---------|--------|----------|
| Quality | Excellent (Pygments) | Good |
| Languages | 500+ | ~80 |
| External deps | Python + Pygments | None |
| Shell-escape | Required | Not needed |
| Speed | Slower | Fast |

### Recommendation
- **Beautiful theme**: `minted` + `tcolorbox` (best quality)
- **Performance theme**: `listings` (no external deps, fast)
- **Fallback**: detect if Pygments is available, use minted; otherwise listings

## Architectural Decisions

### Theme Design
- Two themes: "Beautiful" (minted + tcolorbox + KOMA-Script) and "Performance" (listings + minimal packages)
- Both should support: title page, tables, code listings, math, bibliography
- Beautiful: full KOMA-Script + fontspec + microtype + tcolorbox + minted
- Performance: minimal packages, no fontspec, use listings, avoid heavy TikZ

### Helper Scripts
- `scripts/setup-env.sh` — install portable TeX Live + Python + Pygments
- Helper scripts needed: project init, compile, clean, word count
- Lua scripts: compile time, PDF size, word/page/figure/table counts
