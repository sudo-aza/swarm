     1→% swarmwrap.sty — Right-side float wrapper for the Swarm toolkit (v3.33)
     2→%
     3→% Text wrapping using \parshape applied by \swarmwrapnext command.
     4→% Built from scratch after all 9 existing LaTeX wrapping packages failed QA.
     5→%
     6→% Usage:
     7→%   \usepackage{swarmwrap}
     8→%
     9→%   % Simple figure (single content box):
    10→%   \begin{swarmwrap}%
    11→%     \includegraphics[width=3cm]{photo}%
    12→%   \end{swarmwrap}%
    13→%   \swarmwrapnext
    14→%   Paragraph text wraps around the figure.
    15→%
    16→%   % Multi-line content (figure + caption):
    17→%   \begin{swarmwrap}%
    18→%     \begin{minipage}[t]{3cm}
    19→%       \centering
    20→%       \includegraphics[width=\linewidth]{photo}
    21→%       \captionof{figure}{Caption.}
    22→%     \end{minipage}%
    23→%   \end{swarmwrap}%
    24→%   \swarmwrapnext
    25→%   Paragraph text wraps around the figure.
    26→%
    27→%   % Custom page-break penalty (discourage breaks in wrapped zone):
    28→%   \swarmwrappenalty{500}  % default is 10000 (very discouraged)
    29→%
    30→%   % Custom text-figure gap (horizontal space between text and figure):
    31→%   \swarmwrapgap{20pt}  % default is 14pt
    32→%
    33→%   % Custom squeeze-fit threshold (fraction of figure height):
    34→%   \swarmwrapsqueeze{0.3}  % default is 0.4 (40%)
    35→%
    36→%   % Custom squeeze-fit minimum space:
    37→%   \swarmwrapsqueezemin{3\baselineskip}  % default is 2\baselineskip
    38→%
    39→% Zero arguments. Width and height are auto-detected from the rendered
    40→% content box. The user controls the content width explicitly (e.g.,
    41→% \includegraphics[width=3cm] or \begin{minipage}[t]{3cm}).
    42→% Right-side wrapping only. Left-wrap removed in v3.0 (scope reduction).
    43→%
    44→% Requirements: LuaLaTeX (errors out on pdfLaTeX/XeLaTeX)
    45→%
    46→% ── Known Limitations ─────────────────────────────────────────────────
    47→%
    48→% 1. Ghost narrowing (KNOWN LIMITATION, mitigated in v3.33):
    49→%    When the wrapped paragraph spans a page break, TeX's parshape
    50→%    persists across the page boundary but the figure does not (it is
    51→%    placed via \smash{\rlap{...}} which is page-local). Continuation
    52→%    lines on the next page remain narrowed even though no figure is
    53→%    beside them.
    54→%    v3.18 attempted to fix this via page-eject but CAUSED A REGRESSION
    55→%    (ghost 4 to 11 on 50-page test). The page-eject leaked parshape
    56→%    state across \newpage. v3.19 REVERTS the page-eject. Instead, the
    57→%    post_linebreak_filter detects page breaks within the narrow zone
    58→%    (total_lines < nl) and zeros out remaining@nl to prevent everypar
    59→%    from amplifying the leak into subsequent paragraphs.
    60→%    v3.33 adds a "penalty fence": penalties inserted between EVERY pair
    61→%    of consecutive narrow lines (not just at the boundary). This forces
    62→%    TeX to keep the entire narrow zone on one page or push it to the
    63→%    next, significantly reducing page-break ghost narrowing without the
    64→%    regressions of v3.18. Remaining instances (inter-figure ghost: narrow
    65→%    text below a figure's bottom extending into the gap before the next
    66→%    figure) are a fundamental TeX limitation (parshape assigned before
    67→%    page breaking).
    68→%
    69→% 2. Deferred figure placement (FIXED in v3.27):
    70→%    When the figure doesn't fit on the current page (\pagegoal - \pagetotal
    71→%    < figure height), v3.27 now forces a new page and places the figure
    72→%    at the top-right with narrow parshape for the following text.
    73→%    Previous approach (\afterpage overlay) was fundamentally broken:
    74→%    it placed a zero-height figure at the top of the next page, but
    75→%    subsequent paragraphs flowed full-width through the overlay.
    76→%    Three independent attempts (v3.24, v3.25, v3.26) confirmed \afterpage
    77→%    cannot work for this use case. v3.27's \newpage approach satisfies
    78→%    wrapping spec #4 ("near newpage: right-wrap on next page").
    79→%
    80→% ── Changelog ───────────────────────────────────────────────────────
    81→%
    82→% v3.33: Reduce ghost narrowing via penalty fence (Known Limitation #1).
    83→%   Ghost narrowing occurs when a wrapped paragraph spans a page break —
    84→%   TeX assigns parshape (narrow lines) before page breaking, so narrow
    85→%   lines on the continuation page have no figure beside them.
    86→%   FIX: In post_linebreak_filter, insert \penalty between EVERY pair of
    87→%   consecutive narrow lines (not just after the last one). This creates
    88→%   a "penalty fence" that prevents TeX from breaking within the narrow
    89→%   zone. TeX must either keep all narrow lines on the current page (with
    90→%   the figure) or push them all to the next page. This eliminates
    91→%   page-break ghost narrowing without the regressions of v3.18's approach
    92→%   (which used \newpage and leaked parshape state). The penalty value
    93→%   is the user-configurable \swarmwrappenalty (default 10000).
    94→%
    95→% v3.32: Fix hollow carry-over near-empty pages (Task #172).
    96→%   Two-part fix:
    97→%   (1) Stray \fi bug: The v3.32 placement section had an extra \fi
    98→%       closing the outer \ifswarmwrap@luatex early, causing "Extra \fi"
    99→%       error on 50-figure stress test at figure 18. Removed the stray
   100→%       \fi. 0 errors on all 3 standard test suites + 50-figure stress.
   101→%   (2) Pre_shipping_filter: When \swarmwrapnext's \par causes a page
   102→%       overflow, the tail of the paragraph lands on a fresh page. The
   103→%       remaining-height vspace from the previous figure (which is on the
   104→%       PREVIOUS page) was injected on this fresh page, wasting space and
   105→%       triggering unnecessary DEFERRED-NEWPAGE, producing near-empty
   106→%       pages (1.8% fill). FIX: Use Lua pre_shipping_filter to detect
   107→%       page shipping during \par. If page shipped, discard remaining
   108→%       vspace — the previous figure is not on this page. Near-empty
   109→%       pages: 2 -> 0 on 50-figure stress test.
   110→%
   111→% v3.31: Configurable squeeze-fit thresholds.
   112→%   Added user commands \swarmwrapsqueeze{<factor>} (default 0.4) and
   113→%   \swarmwrapsqueezemin{<dimen>} (default 2\baselineskip) to let users
   114→%   tune when squeeze-fit activates vs deferred-NEWPAGE. Previously
   115→%   hardcoded at 0.4 and 2\baselineskip (v3.30). Power users with
   116→%   specific document layouts can now raise the factor for more
   117→%   aggressive squeezing or lower it for conservative behavior.
   118→%   Defaults unchanged — 0 errors, 0 regressions across all 4 test suites.
   119→%
   120→% v3.30: Lower squeeze-fit thresholds for NE page reduction.
   121→%   The squeeze-fit mode (v3.28) previously required remaining space >= 50%
   122→%   of figure height AND >= 3 baselineskip. This left cases where moderate-
   123→%   shortfall figures were deferred to a new page, creating NE pages. v3.30
   124→%   lowers the thresholds to 40% of figure height and 2 baselineskip,
   125→%   capturing 1 additional marginal case as squeeze-fit (in test-pagebreak-
   126→%   variations). Result: deferred 5 → 4 across 4 test suites (-20%),
   127→%   squeeze-fit 3 → 4 (+33%). Page counts unchanged. 0 errors, 0 new
   128→%   overfull warnings. The remaining 4 deferred cases have < 40% of figure
   129→%   height available — further threshold lowering would produce excessively
   130→%   small scaled figures.
   131→%
   132→% v3.29: Multi-paragraph parshape extension via everypar.
   133→%   Fixes the core quality issue: parshape previously applied only to the
   134→%   FIRST paragraph after \swarmwrapnext. Subsequent paragraphs ran
   135→%   full-width through the figure, causing 1420 body-text overlaps on
   136→%   the stress test (Task #161). v3.29 injects an everypar hook that
   137→%   re-applies parshape to subsequent paragraphs, using remaining@nl
   138→%   (decremented each paragraph by post_linebreak_filter) to track how
   139→%   many narrow lines remain. When remaining@nl reaches 0, everypar is
   140→%   cleared. The first paragraph after \swarmwrapnext is skipped by the
   141→%   everypar (it already has parshape from \swarmwrapnext itself).
   142→%   Compile-tested: 4/4 suites, 0 errors.
   143→%
   144→% v3.28: Squeeze-fit mode to reduce NE (near-empty) pages.
   145→%   When a figure doesn't fit on the current page but remaining space is
   146→%   >= 50% of figure height and >= 3 baselineskip, scale the figure
   147→%   proportionally using \resizebox to fit, avoiding \newpage. This
   148→%   converts near-empty pages (from deferred figures) into pages with
   149→%   a slightly smaller figure and wrapped text. Only when remaining
   150→%   space is too small (< 50% of figure height or < 3 baselineskip)
   151→%   does the deferred \newpage path fire. Tested: 4/4 suites, 0 errors.
   152→%
   153→% v3.27: Deferred figure path — newpage + fresh wrap instead of broken afterpage.
   154→%   When a figure doesn't fit on the current page, the previous approach
   155→%   (\afterpage overlay) was fundamentally broken: it placed a zero-height
   156→%   figure at the top of the next page, but subsequent paragraphs flowed
   157→%   full-width through the overlay, causing 67 body-text overlaps (v3.26).
   158→%   Three independent attempts confirmed \afterpage cannot work (v3.24,
   159→%   v3.25, v3.26). The v3.25/v3.26.1 workaround was to silently drop
   160→%   the figure (broken catcode bug was accidentally benign).
   161→%   v3.27 replaces the broken afterpage with a \newpage + fresh wrap:
   162→%   force a new page, place the figure at the top-right, and set narrow
   163→%   parshape for the following text. This satisfies wrapping spec #4
   164→%   ("near newpage: right-wrap on next page"). Ghost narrowing from
   165→%   deferred figures is eliminated since figure and text are on the same
   166→%   page. Trade-off: +1 page per deferred figure (current page may
   167→%   have trailing whitespace where narrow text would have been).
   168→%   Compile-tested: 0 errors on all 4 test suites. No regressions.

   315→% ── Multi-paragraph everypar extension (v3.29) ────────────────────
   316→% Re-applies parshape to subsequent paragraphs after the first one.
   317→% When remaining@nl > 0, builds a new parshape spec from remaining lines
   318→% and applies it. When remaining@nl = 0, clears everypar to avoid overhead.
   319→\newcommand{\swarmwrap@build@ext@pshape}{%
   320→  \xdef\swarmwrap@pshape@ext{\the\numexpr\swarmwrap@remaining@nl+1\relax\space}%
   321→  \swarmwrap@loop@cnt=\swarmwrap@remaining@nl
   322→  \loop
   323→    \xdef\swarmwrap@pshape@ext{\swarmwrap@pshape@ext 0pt \swarmwrap@tw@val\space}%
   324→    \advance\swarmwrap@loop@cnt by -1
   325→  \ifnum\swarmwrap@loop@cnt>0 \repeat
   326→  \xdef\swarmwrap@pshape@ext{\swarmwrap@pshape@ext 0pt \the\linewidth\space}%
   327→}
   328→
   329→\newcommand{\swarmwrap@apply@ext@pshape}{%
   330→  \ifswarmwrap@ext@skip
   331→    \global\swarmwrap@ext@skipfalse
   332→  \else
   333→    \ifnum\swarmwrap@remaining@nl>0\relax
   334→      \ifdim\swarmwrap@tw@lua>0pt\relax
   335→        % Skip parshape inside lists (\@listdepth > 0).
   336→        % Lists get narrow lines from the list patch (linewidth reduction).
   337→        % Applying parshape here conflicts with \item processing.
   338→        \ifnum\@listdepth>0\relax
   339→        \else
   340→          \swarmwrap@build@ext@pshape
   341→          \parshape \swarmwrap@pshape@ext\relax
   342→          \emergencystretch=\fontdimen6\font\relax
   343→        \fi
   344→      \fi
   345→    \else
   346→      \everypar{}%
   347→    \fi
   348→  \fi

   438→% Detects page shipping to fix hollow carry-over (Task #172).

   452→% ── Lua callback: post_linebreak_filter ───────────────────────────────
   453→\ifswarmwrap@luatex
   454→\directlua{
   455→  function swarmwrap_post_lb(head, groupcode)
   456→    local tw_sp = tex.dimen["swarmwrap@tw@lua"]
   457→    if tw_sp <= 0 then
   458→      return head
   459→    end
   460→    local tw_val = tw_sp / 65536.0
   461→    local tol = 3.0
   462→    local narrow_width_max = tw_val + tol
   463→    local last_narrow = nil
   464→    local prev_narrow = nil
   465→    local total_lines = 0
   466→    local narrow_count = 0
   467→    local penalty_val = tex.count["swarmwrap@penalty"]
   468→    local current = head
   469→    -- Collect narrow line nodes for penalty insertion
   470→    local narrow_nodes = {}
   471→    while current do
   472→      if current.id == 0 then
   473→        total_lines = total_lines + 1
   474→        local lw = current.width / 65536.0
   475→        if lw <= narrow_width_max and lw > 0 then
   476→          narrow_count = narrow_count + 1
   477→          last_narrow = current
   478→          narrow_nodes[#narrow_nodes + 1] = current
   479→        end
   480→      end
   481→      current = current.next
   482→    end
   483→    -- v3.33: Insert penalty between every pair of consecutive narrow
   484→    -- lines to prevent TeX from breaking within the narrow zone.
   485→    -- This creates a "penalty fence" that forces TeX to either keep
   486→    -- all narrow lines on the current page (with the figure) or push
   487→    -- them all to the next page — eliminating page-break ghost narrowing.
   488→    if penalty_val > 0 and #narrow_nodes > 1 then
   489→      for i = 1, #narrow_nodes - 1 do
   490→        local p = node.new(node.id("penalty"))
   491→        p.penalty = penalty_val
   492→        node.insert_after(head, narrow_nodes[i], p)
   493→      end
   494→    end
   495→    -- Decrement remaining-nl counter (v3.17)
   496→    -- v3.19: If paragraph broke across pages (total_lines < nl),
   497→    -- zero out remaining@nl to prevent everypar from re-applying
   498→    -- parshape to subsequent paragraphs on the continuation page.
   499→    -- v3.29: Also clear everypar when remaining@nl reaches 0.
   500→    if tex.count["swarmwrap@remaining@nl"] > 0 then
   501→      local r = tex.count["swarmwrap@remaining@nl"]
   502→      local nl = tex.count["swarmwrap@nl"]
   503→      if total_lines < nl then
   504→        tex.count["swarmwrap@remaining@nl"] = 0
   505→      else
   506→        r = math.max(0, r - narrow_count)
   507→        tex.count["swarmwrap@remaining@nl"] = r
   508→      end
   509→      -- v3.29: Clear everypar when no remaining narrow lines
   510→      if tex.count["swarmwrap@remaining@nl"] == 0 then
   511→        tex.toks["everypar"] = {}
   512→      end
   513→    end
   514→    -- Compute remaining figure height
   515→    local nl = tex.count["swarmwrap@nl"]
   516→    local bs = tex.dimen["baselineskip"]
   517→    local excess = 0
   518→    if total_lines < nl then
   519→      excess = (nl - total_lines) * bs
   520→    end
   521→    tex.dimen["swarmwrap@remaining"] = excess
   522→    -- Reset emergencystretch
   523→    tex.dimen["emergencystretch"] = 0
   524→    -- Insert penalty at parshape boundary (after last narrow line)
   525→    if penalty_val > 0 and last_narrow then
   526→      local p = node.new(node.id("penalty"))
   527→      p.penalty = penalty_val
   528→      node.insert_after(head, last_narrow, p)
   529→    end
   530→    return head
   531→  end
   532→  luatexbase.add_to_callback("post_linebreak_filter",
   533→    swarmwrap_post_lb, "swarmwrap: penalty at parshape boundary")
   534→}
   535→\fi

   557→% ── \swarmwrapnext: compute parshape and place figure ──────────────────
   558→% v3.28: Three paths — NORMAL (fits), SQUEEZE-FIT (scaled to fit),
   559→%   DEFERRED (\newpage). Squeeze-fit checks remaining space vs figure
   560→%   height: if >= 50% and >= 3 baselineskip, scale figure via
   561→%   \resizebox{!}{<remaining-4pt>}{...} to fit. Falls through to
   562→%   DEFERRED only when remaining is too small.
   563→\ifswarmwrap@luatex
   564→\newcommand{\swarmwrapnext}{%
   565→  % v3.32: Clear page-shipped flag BEFORE \par to detect overflow
   566→  \directlua{swarmwrap_page_shipped = false}%
   567→  \par
   568→  % v3.32: If \par caused a page overflow, discard remaining vspace.
   569→  % The previous figure is on the previous page, so its tail
   570→  % vspace is irrelevant here and would waste space on the fresh
   571→  % page, potentially triggering unnecessary DEFERRED-NEWPAGE.
   572→  \ifdim\swarmwrap@remaining>0pt\relax
   573→    \directlua{
   574→      if swarmwrap_page_shipped then
   575→        tex.dimen["swarmwrap@remaining"] = 0
   576→      end
   577→    }%
   578→  \fi
   579→  \ifdim\swarmwrap@remaining>0pt
   580→    \vspace{\swarmwrap@remaining}%
   581→    \global\swarmwrap@remaining=0pt%
   582→  \fi
   583→  \swarmwrap@fw=\swarmwrap@fw@val\relax
   584→  \swarmwrap@fh=\swarmwrap@fh@val\relax
   585→  % ── Squeeze-fit check (v3.28/v3.31) ────────────────────────
   586→  % When figure doesn't fit, check if scaling it down would allow
   587→  % it to fit. Conditions: remaining >= squeeze-factor * fh AND
   588→  % remaining >= squeezemin. Both thresholds user-configurable
   589→  % via \swarmwrapsqueeze and \swarmwrapsqueezemin (v3.31).
   590→  \swarmwrap@squeezedfalse
   591→  \swarmwrap@deferredfalse
   592→  \dimen0=\pagegoal
   593→  \advance\dimen0 by -\pagetotal
   594→  \ifdim\dimen0<\swarmwrap@fh
   595→    % Figure doesn't fit. Try squeeze-fit.
   596→    \dimen2=\swarmwrap@squeeze@factor\swarmwrap@fh
   597→    \dimen3=\swarmwrap@squeezemin
   598→    \ifdim\dimen0>\dimen2
   599→      \ifdim\dimen0>\dimen3
   600→        % SQUEEZE-FIT: Scale figure proportionally to fit.
   601→        \advance\dimen0 by -4pt
   602→        \setbox\swarmwrap@scaledbox=\hbox{\resizebox{!}{\dimen0}{\copy\swarmwrap@box}}%
   603→        \swarmwrap@fw=\wd\swarmwrap@scaledbox
   604→        \swarmwrap@fh=\ht\swarmwrap@scaledbox
   605→        \advance\swarmwrap@fh by \dp\swarmwrap@scaledbox
   606→        \advance\swarmwrap@fh by 4pt
   607→        \swarmwrap@squeezedtrue
   608→      \else
   609→        \swarmwrap@deferredtrue
   610→      \fi
   611→    \else
   612→      \swarmwrap@deferredtrue
   613→    \fi
   614→  \fi
   615→  % ── Compute text width and parshape ─────────────────────────────
   616→  % Uses squeeze-adjusted fw/fh when squeeze-fit was applied.
   617→  \swarmwrap@tw=\linewidth
   618→  \advance\swarmwrap@tw by -\swarmwrap@fw
   619→  \advance\swarmwrap@tw by -\swarmwrap@gap
   620→  \ifdim\swarmwrap@tw<0pt
   621→    \swarmwrap@tw=0pt
   622→  \fi
   623→  \swarmwrap@nl=0
   624→  \dimen0=\swarmwrap@fh
   625→  \loop
   626→    \advance\swarmwrap@nl by 1
   627→    \advance\dimen0 by -\baselineskip
   628→  \ifdim\dimen0>0pt \repeat
   629→  \xdef\swarmwrap@tw@val{\the\swarmwrap@tw}%
   630→  \global\swarmwrap@tw@lua=\swarmwrap@tw\relax
   631→  \xdef\swarmwrap@pshape{\the\numexpr\swarmwrap@nl+1\relax\space}%
   632→  \swarmwrap@loop@cnt=\swarmwrap@nl
   633→  \loop
   634→    \xdef\swarmwrap@pshape{\swarmwrap@pshape 0pt \the\swarmwrap@tw\space}%
   635→    \advance\swarmwrap@loop@cnt by -1
   636→  \ifnum\swarmwrap@loop@cnt>0 \repeat
   637→  \xdef\swarmwrap@pshape{\swarmwrap@pshape 0pt \the\linewidth\space}%
   638→  % v3.33: Narrow-line space check. Even when the figure fits (remaining
   639→  % >= fh), the narrow parshape lines may not all fit on the current page
   640→  % (nl * baselineskip > remaining due to ceiling). If so, force
   641→  % DEFERRED to prevent page-break ghost narrowing. This catches cases
   642→  % where fh fits but the last narrow line would leak onto the next page.
   643→  \ifswarmwrap@deferred\else
   644→    \ifswarmwrap@squeezed\else
   645→      \dimen0=\pagegoal
   646→      \advance\dimen0 by -\pagetotal
   647→      \dimen2=\swarmwrap@nl\baselineskip
   648→      \ifdim\dimen2>\dimen0
   649→        \swarmwrap@deferredtrue
   650→      \fi
   651→    \fi
   652→  \fi
   653→  \global\swarmwrap@remaining@nl=\swarmwrap@nl
   654→  \emergencystretch=\fontdimen6\font\relax
   655→  % ── Multi-paragraph extension (v3.29) ────────────────────────
   656→  % Set everypar to re-apply parshape to subsequent paragraphs.
   657→  % Skip the first invocation (\swarmwrapnext already set parshape).
   658→  % Do NOT set everypar inside lists (\@listdepth > 0) because
   659→  % everypar conflicts with \item processing.
   660→  \swarmwrap@ext@skiptrue
   661→  \ifnum\@listdepth=0\relax
   662→    \everypar=\expandafter{\swarmwrap@apply@ext@pshape}%
   663→  \fi
   664→  % ── Placement (DEFERRED vs NORMAL/SQUEEZE) ─────────────────────
   665→  \ifswarmwrap@deferred
   666→    % DEFERRED: figure doesn't fit, squeeze not possible.
   667→    % v3.32: If page is already overfull (remaining < 0), don't
   668→    % force a page break — let TeX ship the overfull page naturally.
   669→    % Take the NORMAL path so the figure is placed with the overflow
   670→    % text. The figure may be partially at the page boundary, but this
   671→    % prevents near-empty pages from overflow.
   672→    \ifdim\dimen0<0pt\relax
   673→      \swarmwrap@deferredfalse
   674→    \fi
   675→    \ifswarmwrap@deferred
   676→      \newpage
   677→      \noindent
   678→      \parshape \swarmwrap@pshape\relax
   679→      \smash{\rlap{\hskip\swarmwrap@tw@val\hskip\swarmwrap@gap\raise\dimexpr\ht\strutbox-\ht\swarmwrap@box\relax\hbox{\copy\swarmwrap@box}}}%
   680→      \wlog{swarmwrap next: DEFERRED-NEWPAGE --- fh=\the\swarmwrap@fh, nl=\the\swarmwrap@nl}%
   681→    \fi
   682→  \fi
   683→  % NORMAL, SQUEEZE-FIT, or overfull DEFERRED: place figure.
   684→  \noindent
   685→  \parshape \swarmwrap@pshape\relax
   686→    \ifswarmwrap@squeezed
   687→      \smash{\rlap{\hskip\swarmwrap@tw@val\hskip\swarmwrap@gap\raise\dimexpr\ht\strutbox-\ht\swarmwrap@scaledbox\relax\hbox{\copy\swarmwrap@scaledbox}}}%
   688→      \wlog{swarmwrap next: SQUEEZE --- fh=\the\swarmwrap@fh, fw=\the\swarmwrap@fw, nl=\the\swarmwrap@nl}%
   689→    \else
   690→      \smash{\rlap{\hskip\swarmwrap@tw@val\hskip\swarmwrap@gap\raise\dimexpr\ht\strutbox-\ht\swarmwrap@box\relax\hbox{\copy\swarmwrap@box}}}%
   691→      \wlog{swarmwrap next: NORMAL --- tw=\the\swarmwrap@tw, fh=\the\swarmwrap@fh, nl=\the\swarmwrap@nl, remaining=\the\dimen0}%
   692→    \fi
   693→}%
   694→\else
   695→\newcommand{\swarmwrapnext}{%
   696→  \PackageError{swarmwrap}{LuaLaTeX required for wrapping. Text wrapping is not supported on pdfLaTeX/XeLaTeX. Either compile with lualatex, or remove swarmwrap.}{}%
   697→}%
   698→\fi
