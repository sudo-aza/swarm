   853→### Ghost Narrowing Mitigation
   854→- **Root cause**: TeX processes parshape during line breaking, BEFORE page breaking determines which lines go on which page. Parshape lines that end up on the continuation page have no figure beside them.
   855→- **`post_linebreak_filter`**: Operates before page assignment — cannot detect which page lines will go to. **Cannot solve**.
   856→- **`buildpage_filter`**: Could reject page breaks leaving orphan narrowed lines, but risks infinite loops and poor page breaks. **Too risky**.
   857→- **`\localrightbox`**: Per-paragraph primitive, not per-line. Cannot conditionally show figure only on narrowed lines. **Not useful**.
   858→- **Two-pass Lua approach**: Record page breaks in pass 1, adjust parshape in pass 2. Theoretically complete but fragile, slow, and circular (changing parshape changes line breaking which changes page breaks). **Research project only**.
   859→- **Simple mitigation**: `\swarmwrappenalty{N}` inserts penalty after narrowed zone to discourage page breaks.
   860→- **Recommendation**: Accept and document (cosmetic only, no overlap). Prioritize task #143.
