   785→### Ghost Narrowing Mitigation
   786→- **Root cause**: TeX processes parshape during line breaking, BEFORE page breaking determines which lines go on which page. Parshape lines that end up on the continuation page have no figure beside them.
   787→- **`post_linebreak_filter`**: Operates before page assignment — cannot detect which page lines will go to. **Cannot solve**.
   788→- **`buildpage_filter`**: Could reject page breaks leaving orphan narrowed lines, but risks infinite loops and poor page breaks. **Too risky**.
   789→- **`\localrightbox`**: Per-paragraph primitive, not per-line. Cannot conditionally show figure only on narrowed lines. **Not useful**.
   790→- **Two-pass Lua approach**: Record page breaks in pass 1, adjust parshape in pass 2. Theoretically complete but fragile, slow, and circular (changing parshape changes line breaking which changes page breaks). **Research project only**.
   791→- **Simple mitigation**: `\swarmwrappenalty{N}` inserts penalty after narrowed zone to discourage page breaks.
   792→- **Recommendation**: Accept and document (cosmetic only, no overlap). Prioritize task #143.

   794→> Full details in `notes/2026-05-18-ghost-narrowing-research.md`
