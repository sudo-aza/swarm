   833Ôćĺ### Ghost Narrowing Mitigation
   834Ôćĺ- **Root cause**: TeX processes parshape during line breaking, BEFORE page breaking determines which lines go on which page. Parshape lines that end up on the continuation page have no figure beside them.
   835Ôćĺ- **`post_linebreak_filter`**: Operates before page assignment ÔÇö cannot detect which page lines will go to. **Cannot solve**.
   836Ôćĺ- **`buildpage_filter`**: Could reject page breaks leaving orphan narrowed lines, but risks infinite loops and poor page breaks. **Too risky**.
   837Ôćĺ- **`\localrightbox`**: Per-paragraph primitive, not per-line. Cannot conditionally show figure only on narrowed lines. **Not useful**.
   838Ôćĺ- **Two-pass Lua approach**: Record page breaks in pass 1, adjust parshape in pass 2. Theoretically complete but fragile, slow, and circular (changing parshape changes line breaking which changes page breaks). **Research project only**.
   839Ôćĺ- **Simple mitigation**: `\swarmwrappenalty{N}` inserts penalty after narrowed zone to discourage page breaks.
   840Ôćĺ- **Recommendation**: Accept and document (cosmetic only, no overlap). Prioritize task #143.

   842Ôćĺ> Full details in `notes/2026-05-18-ghost-narrowing-research.md`
